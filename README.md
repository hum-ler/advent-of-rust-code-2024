# advent-of-rust-code-2024

## Usage

```bash
cargo run <DAY_NUMBER> <RUN_CODE>
```
where `RUN_CODE` can be:
- `e1` -- runs example 1
- `e2` -- runs example 2
- `p1` -- runs part 1
- `p2` -- runs part 2
